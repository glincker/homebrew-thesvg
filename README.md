# homebrew-thesvg

Homebrew tap for [theSVG](https://thesvg.org).

## Install

```bash
brew tap glincker/thesvg
brew install thesvg
```

## What it installs

`@thesvg/cli`, a Node-based CLI for working with the theSVG icon library.

```bash
thesvg add github
thesvg search "ai"
```

## Updating

```bash
brew update
brew upgrade thesvg
```

The formula here is kept in sync with each `@thesvg/cli` release on npm via an automated workflow in [glincker/thesvg](https://github.com/glincker/thesvg).

## Issues, suggestions

Open an issue at https://github.com/glincker/thesvg/issues. This repo only holds the formula.
